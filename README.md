# Netflix Clone

A responsive Netflix-inspired web application built using HTML, CSS, and JavaScript. The project features a modern UI, smooth navigation, interactive movie cards with hover effects, and a fully responsive design.

Features

* Responsive Navbar
* Hero Banner Section
* TV Shows, Movies & My List Sections
* Interactive Movie Cards
* Hover Effects with Movie Details
* Smooth Scrolling Navigation
* Mobile-Friendly Design

Technologies Used

* HTML5
* CSS3
* JavaScript
